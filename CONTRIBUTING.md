# Contributing

Thanks for your interest in contributing.

## Reporting Bugs

Open a [bug report](https://github.com/liamvibecodes/plex-lights/issues/new?template=bug-report.yml) with:
- What happened vs what you expected
- Your config (redact API keys)
- Relevant log output

## Feature Requests

Open a [feature request](https://github.com/liamvibecodes/plex-lights/issues/new?template=feature-request.yml) with:
- What you want and why
- Any examples from similar tools

## Pull Requests

1. Fork and create a branch
2. Test your changes with a real Tautulli webhook
3. Keep the PR focused on one thing
4. Submit against `main`
